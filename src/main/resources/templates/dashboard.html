Bienvenido/a a ERP Hotel

Estamos encantados de presentarle nuestro sistema de gestión empresarial (ERP) especialmente diseñado para hoteles. Nuestro software ofrece una solución completa y personalizable para ayudar a administrar y optimizar todas las operaciones de su hotel.

Funcionalidades

Reservas

En el apartado de reservas el trabajador puede añadir reservas con variadas de funcionalides, a parte de visualizarlas.

Facturación

En este apartado se podrá gestionar tanto las facturas, como las líneas que contienen, para así tener un registro de facturación de cada reserva que se hace en el hotel.

Habitaciones

Este apartado esta para gestionar las habitaciónes, podremos agregar nuevas habitaciónes, asi como controlar el estado en el que se encuentran.

Clientes

Dentro del módulo de clientes, es posible visualizar tanto a los clientes como a los trabajadores en una lista, desde la cual se puede acceder a sus respectivos formularios de edición y eliminarlos en caso de ser necesario. Además, también se encuentra disponible la opción de agregar nuevos trabajadores.

Usuarios

Este sistema ERP dispone de un sistema de seguridad de usuarios que limita el acceso de los trabajadores en función de sus permisos y roles. Dependiendo del trabajo que desempeñen, los usuarios pueden acceder a ciertas áreas y recursos, mientras que otros estarán restringidos.
